# Web Scraping - Proyecto de Extracción de Datos

Este proyecto es un **script de web scraping** desarrollado con **Node.js** y **Puppeteer**, que recopila información de una página web específica y la guarda en un archivo `data.json` de manera estructurada.

## Instalación

```bash
git clone https://github.com/IliasVilux/web-scraping.git
cd web-scraping
npm install
```

## Uso

```bash
node index.js
```
Después de la ejecución, encontrarás el archivo `data.json` con todos los datos recopilados.
